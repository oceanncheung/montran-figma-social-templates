# Figma Workflow

## File Structure

The primary file is:

https://www.figma.com/design/9pzw3P4vngkkAiitYtcgwn/Montran-Social-Media-Templates

Important pages:

- `Social Templates`: source of reusable layout ideas and formats.
- Monthly pages such as `Apr 2026`, `Mar 2026`, `Feb 2026`: final production pages.
- `Reference`: use only for visual/context reference when helpful.
- `Awards`: ignore unless the user explicitly asks for award-related content.

## Placement

Use the planned content date from Notion to choose the monthly page and row. If the actual posting date differs from the planned slot, keep the design in the planned Figma slot unless the user asks to move it.

Example: a post planned for February 24 but published later in March can remain on the `Feb 2026` page under the `24` row.

## Row Structure

Maintain this order in each content row:

`EN -> EN slides -> ES -> ES slides -> FR -> FR slides -> Web`

If only English is requested, place the English slides after `EN` and preserve the `ES`, `FR`, and `Web` placeholders.

## Template Use

Use `Social Templates` for available formats, but also browse prior monthly pages to find layouts that better match the content. Previous real posts are often better references for visual hierarchy, image use, and carousel pacing.

Recommended process:

1. Read the Notion page and identify content type, language requirements, and planned date.
2. Locate the correct monthly page and day row in Figma.
3. Review nearby monthly examples and relevant template frames.
4. Build English first.
5. Validate the visual layout and row order.
6. Duplicate approved English slides for Spanish and French when requested.

## Figma Editing Notes

- Use soft returns for line breaks inside a text object.
- Avoid overwriting unrelated existing content in a row.
- If a row for the date is already occupied, use the next available row for the same day when present.
- After duplicating slides, reorder the row layers so auto-layout displays the correct sequence.
- When replacing images, only replace images on the intended slides.
