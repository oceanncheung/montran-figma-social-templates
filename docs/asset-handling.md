## Cloud-Sync Issue

Images in OneDrive or iCloud-backed folders may appear in Finder but fail to import into Figma if the original file is not fully downloaded locally.

If image loading fails:

1. Confirm the image is downloaded locally.
2. If needed, copy the image to a local temporary folder outside the cloud-synced path.
3. Import the local copy into Figma.
4. Keep the same crop, mask, overlay, and slide treatment unless the user asks for a new visual direction.

## Replacement Rules

- Replace only the images the user asks to update.
- Do not add images to text-only slides unless requested.
- Preserve existing masks and overlay colors where possible.
- Check the image crop visually after replacement.
- If a sharper image imports poorly into a masked frame, choose the version that sits best in the design.

## Image Choice

Prefer images that fit the content theme:

- banking and commercial finance
- payment infrastructure
- business people in professional settings
- technology and connectivity
- regional relevance when available
- subtle motion or architecture when the post is about infrastructure

Avoid images that feel generic, overly staged, or unrelated to the argument.
